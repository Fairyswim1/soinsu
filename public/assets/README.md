# Asset replacement folders

The game runs without binary assets by generating Phaser fallback candy textures and Web Audio effects.

Drop replacement files into these folders using the names listed in `src/game/config/assetManifest.ts`.

- `candies/`
- `specials/`
- `blockers/`
- `backgrounds/`
- `ui/`
- `effects/`
- `audio/`
